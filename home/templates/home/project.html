In Retrospect

I created this website using Django and HTMX. Many challenges arose from beginner mistakes. This includes setting the project up in a global environment which caused many dependency issues. Another error was not keeping a clean codebase which is a common mistake when using HTMX, as everything is built within the same application to allow for smoother transitions. It's purely front-end. Expect future projects to have added layers of complexity, as well as functionality.
